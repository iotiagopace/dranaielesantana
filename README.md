## Observações
- O template original usa a fonte Kollektif. Como ela não está disponível no Google Fonts e não podemos embutir arquivos de fonte proprietários, usei `Plus Jakarta Sans` como fallback próximo e deixei `Kollektif` como primeira opção no CSS caso você tenha licença/arquivo da fonte.
- Os ícones dos serviços foram recriados em SVG inline, mantendo a lógica de iconografia odontológica do template.
- O número do WhatsApp está temporário como `5517999999999`. Substituir pelo número oficial da clínica no `index.html`.
